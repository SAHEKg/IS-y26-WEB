Консольный прогноз погоды

C++
Консольное приложение на C++, отправляющее запросы на API прогноза погоды и отрисовывающее результаты в виде таблицы с ASCII-art артинками для разных погодных условий. Есть возможность менять место и количество дней прогноза

Консольный банкомат

C#, PostgreSQL
Консольное приложение на C#, симулирующее работу банкомата. Написано на гексагональной архитектуре. Можно класть и снимать деньги, смотреть баланс, переводить деньги другим людям. Ошибочные состояния обрабатываются корректно, взаимодействие происходит выбором действий стрелочками клавиатуры
